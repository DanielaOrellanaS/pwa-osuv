🔗 Google Sheets No configurado
Sincroniza automáticamente con un Google Sheet compartido para que todos puedan ver las asistencias.
✅ Una vez configurado, la sincronización ocurre automáticamente al guardar cada ensayo.

📊 Configurar Google Sheets

¿Qué necesitas? Solo una cuenta de Google y 5 minutos. Es completamente gratuito.
Paso 1 — Crea el Google Sheet
Ve a sheets.google.com → crea una hoja nueva → nómbrala "OSUV Asistencia" → comparte el link con quién quieras que lo vea.

Paso 2 — Abre el editor de Scripts
En el Sheet ve a Extensiones → Apps Script. Borra todo el código que aparece.

Paso 3 — Pega este código:
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); data.forEach(function(sheetData) { var sheetName = sheetData.month; var sheet = ss.getSheetByName(sheetName); if (!sheet) sheet = ss.insertSheet(sheetName); sheet.clearContents(); var headers = ["Seccion", "Musico"].concat(sheetData.dates); sheet.appendRow(headers); sheetData.rows.forEach(function(row) { sheet.appendRow(row); }); var hRange = sheet.getRange(1, 1, 1, headers.length); hRange.setBackground("#0f2548"); hRange.setFontColor("#ffffff"); hRange.setFontWeight("bold"); }); return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService.createTextOutput(JSON.stringify({ok:false,error:err.message})).setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { return ContentService.createTextOutput(JSON.stringify({ok:true,msg:"OSUV Script activo"})).setMimeType(ContentService.MimeType.JSON); }
Paso 4 — Despliega el script
Clic en Implementar → Nueva implementación → Tipo: Aplicación web → "Quién tiene acceso": Cualquier usuario → Clic en Implementar.

Paso 5 — Copia la URL
Copia la URL que aparece (empieza con https://script.google.com/macros/s/...) y pégala en el campo de la app.

⚠️ Importante: La primera vez que sincronices, Google pedirá que autorices permisos. Acepta con tu cuenta de Google.
